This project analyzes pizza sales data using SQL to extract business insights such as:
- Best-selling pizzas
- Revenue trends
- Peak order times
- Customer buying behavior

---

## 📂 Dataset Files
- pizzas.csv
- pizza_types.csv
- orders.csv
- order_details.csv

---

## 🛠️ Tools Used
- SQL (MySQL / PostgreSQL)
- Excel (optional)
- Data Analysis Techniques

---

## 📊 Key Business Insights
- Top revenue-generating pizzas
- Most popular pizza sizes
- Peak order hours
- Monthly sales trends
